Задание

1. Создайте русскоязычную версию сайта ru/index.html.

2. Создайте англоязычную версию сайта en/index.html.

3. Создайте index.html в корне проекта для общей информации, в котором должна быть навигация по страницам, добавленных ранее.

4. Настройте GitHub Pages на автоматическую сборку из ветки gh-pages.

5. Реализуйте CI/CD, выполняющий при создании нового релиза публикацию приложения на GitHub Actions. Приложение должно быть доступно по адресу {ваш-адрес-githubpages.io}/ваш￾репозиторий/v{название вашего релиза}/

6. Обеспечить доступность прошлых версий сайта.

7. Оформить отчет, в котором необходимо:

a. Приложить скриншоты настройки GitHub Pages и с описанием

для чего и как будет работать каждая настройка.

b. Скриншот работы CI/CD с описанием каждого шага.

c. Скриншоты web приложения, на которых видно адресную строку браузера.

d. Код pipeline.
